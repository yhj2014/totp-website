打包教程

手动打包教程

Windows EXE 打包

  1. 安装 Python 3.8+ 和 Git
  2. 安装依赖库： 
    pip install pyinstaller pyotp qrcode opencv-python pillow cryptography
  3. 执行打包命令： 
    pyinstaller --onefile --windowed --name=TOTP_Authenticator totp.py
  4. 打包完成后，EXE文件位于 dist/ 目录

Android APK 打包

  1. 安装 Android Studio 并配置 SDK/NDK
  2. 安装 BeeWare 工具： 
    pip install briefcase
  3. 初始化项目： 
    briefcase new
  4. 创建并构建 Android 项目： 
    briefcase create android
briefcase build android
  5. APK文件位于 android/ 目录

使用脚本打包教程

Windows EXE 打包脚本

  1. 下载 build_exe.ps1 脚本
  2. 右键脚本选择"使用 PowerShell 运行"
  3. 等待自动完成：
    • ✓ 自动安装依赖
    • ✓ 自动清理临时文件
    • ✓ 输出到 TOTP_Windows/ 目录
  4. 将整个文件夹分发给用户

Android APK 打包脚本

  1. 确保已配置 Android SDK/NDK
  2. 下载 build_apk.ps1 脚本
  3. 右键脚本选择"使用 PowerShell 运行"
  4. 等待构建完成（约10-20分钟）：
    • ✓ 自动配置项目
    • ✓ 处理依赖关系
    • ✓ 输出到 TOTP_Android/ 目录
注意： 首次运行需接受Android许可证：sdkmanager --licenses
